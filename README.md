# find_

## 사용법
* 특정 파일 검색 시
```
find '찾을 파일/폴더' -'옵션'
```

* 디렉토리 내 모든 파일 출력
```
find
```

## 사용 예시
* 1번 예시
  * 입력
  ```
  find find.c
  ```
  * 출력
  ```
  c 드라이브의 볼륨: C
 볼륨 일련 번호: 7A57-8ADA

 c:\project\find 디렉터리

               1개 파일               2,629 바이트
               0개 디렉터리  404,096,675,840 바이트 남음

  ```
